# Lararunner 
A way to run fast in Laravel Starter


### How to build

Assuming you have *php* and *composer* installed.

Install the packages:

1. `composer install`
2. `php artisan migrate`
3. `php artisan serv`

On Production:
```shell
$ composer install --optimize-autoloader --no-dev
```
